# README – Projet de compilation

## Ce qui fonctionne

- Déclarations : constantes, variables globales et locales
- Expressions : arithmétiques, booléennes, comparaisons, opérateurs logiques
- Instructions : lire, ecrire, affectation (`:=`)
- Structures de contrôle : `si`/`sinon`/`fsi`, `ttq`/`faire`/`fait`, `cond`/`fcond`
- Compilation des procédures : déclaration, paramètres fixe et mod, variables locales, appel

## Ce qui ne fonctionne pas

- **Vérification du nombre de paramètres à l'appel**

- **Vérification du type des paramètres effectifs** 

- **Tests d'erreurs sémantiques**

## Difficultés rencontrées

La compilation du cond et des procédures ont été les parties les plus difficiles
